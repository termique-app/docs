The Termique mobile app is a companion to the desktop. It uses the same API backend and the same encryption: your hosts, groups, snippets, and SSH sessions are available on your phone once you sign in and unlock.

## Platforms

| Platform | Status |
|----------|--------|
| iOS | Available on the App Store |
| Android | Available on Google Play |

## What's available on mobile

- Add, edit, and connect to hosts
- Full SSH terminal sessions
- SFTP file browsing and transfer
- Groups and snippets (create, edit, run)
- Generate or import SSH keys
- AI assistant
- Command log viewing
- Monitoring alert rules, with push notifications for hosts with the monitoring agent installed
- Accept and manage share invites

## Sync

Mobile uses the same API backend as the desktop. Sign in with the same account, enter your master password to unlock, and your data is available immediately.

The master password and DEK derivation are interoperable between desktop and mobile: credentials encrypted on desktop can be decrypted on mobile and vice versa.

## Push notifications

The mobile app supports push notifications for monitoring alerts: when a host with the monitoring agent installed crosses an alert rule's threshold (CPU, RAM, or disk), you get a push notification. Tapping it opens the alert rules screen for that host.

Notification delivery requires the mobile app to be installed and notifications to be enabled in iOS/Android system settings.
